# rpm-httpd22

Apache old line 2.2 package built to be co-existance with new 2.4 version without conflicts.

Primarily for use in 1C:Enterproce web-services and allow also use Apache (httpd) 2.4 on tha same server.

Mos files renamed to have 22 suffix like /etc/httd22, /var/log/httpd22 and so on.
